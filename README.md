Кейс по анализу данных по продажам персональных компьютеров и комплектующих на торговых онлайн площадках
Состав: 
1. Входные данные
2. Карточка проекта, согласованная с Заказчиком
3. Анализ данных, выполненый в Google Colab с комментариями, графиками и аналитической запиской.
4. Дашборд (Power BI) на основе предобработанных данных
